Cron Updates:

  • You are running core module version | This feature requires core module version 1.13.0 or higher from github: https://github.com/damanic/ls1-module-core

Enabling Cron :To enable lemonstands cron feature your system scheduler must be set up to trigger the core cron process at an interval greater or equal to your selected update interval. Example command to add to your scheduler: /usr/local/bin/php -q /ROOT/PATH/TO/SITE/public_html/modules/core/cron.php

Note: When cron is enabled all rate updates will be handled by a background process. If not enabled Lemonstands Currency Converter will check if new rates are required on every request. Requests are usually made on page load which can cause your site to hang if the conversion service becomes unavailable.
