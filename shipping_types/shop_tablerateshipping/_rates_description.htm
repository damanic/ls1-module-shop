• Country Code: Two letter ISO code for a country, or asterisk (*) for any country
  • State Code: As defined in `System->Settings->Countries and States`, or asterisk (*) for any state
  • Postal Code: Asterisk (*) for any postal code. Separate with commas if more than one code. A number range can also be added using brackets. Eg `CODE-[1-10]`
  • City: Asterisk (*) for any city
