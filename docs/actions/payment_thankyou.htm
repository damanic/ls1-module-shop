Use this action to create a "thank you for the payment" page.

Important: this action integrates Google Analytics E-Commerce transaction tracking. Always use a page with this action assigned in the end of payment process if you use Google Analytics E-Commerce features.

Variables

  • $order - represents an order. Use this variable to display order-specific information like order items and totals. An object of the Shop_Order class
  • $items - a list of order items. An object of the Db_DataCollection class. Each element of the collection is an object of the Shop_OrderItem class
